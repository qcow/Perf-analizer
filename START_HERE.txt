✨ KEY FEATURES
───────────────────────────────────────────────────────────────────────────

✅ Single-Page Interface
   - Clean, modern design
   - Dark theme for eye comfort
   - Responsive (desktop & mobile)

✅ Browser Automation
   - Microsoft.Playwright integration
   - Headless Chromium browser
   - Automatic resource capture

✅ Performance Metrics
   - Page load time (milliseconds)
   - Total resource size (KB/MB)
   - Per-asset breakdown
   - Category analysis (JS/CSS/IMG)

✅ Critical Detection
   - Flags large files (>300KB)
   - Identifies slow resources (>500ms)
   - Visual ⚠️ CRITICAL badges

✅ Beautiful Results
   - Summary cards (3 metrics)
   - Category breakdown (4 cards)
   - Waterfall table (detailed list)
   - Status indicators

💻 QUICK COMMANDS
───────────────────────────────────────────────────────────────────────────

Start the app:
  dotnet run --project PerfAnalyzer.csproj

Build without running:
  dotnet build PerfAnalyzer.csproj

Clean & rebuild:
  dotnet clean PerfAnalyzer.csproj && dotnet build PerfAnalyzer.csproj

Run specific profile:
  dotnet run --project PerfAnalyzer.csproj --launch-profile https

Check .NET version:
  dotnet --version
